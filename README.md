# THE NOOK & BREW STUDY CAFE

This is a Study Cafe that provides a variety of study spaces for visitors. From secluded solo spaces to group study areas, the cafe has all sorts of nooks for you!

## Tech Stack

- React and Tailwind CSS for frontend
- Lucide for icons
- JSON server for simulating backend
- Vercel for frontend deployment and Railway for json hosting
